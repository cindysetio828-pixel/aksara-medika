# Aksara Medika
Sistem Pendaftaran Pasien Online berbasis web menggunakan CodeIgniter 3 dan MySQL.

## Deskripsi Project
Aksara Medika adalah sistem pendaftaran pasien online yang dibuat untuk memudahkan proses pendaftaran berobat di rumah sakit.  
Melalui sistem ini, pasien dapat melakukan registrasi akun, mengisi form pendaftaran berobat, memilih dokter spesialis, menentukan jadwal kunjungan, serta melihat status pendaftaran secara online.  
Admin dapat mengelola data pasien, data dokter, data pendaftaran, jadwal pasien, dan laporan pendaftaran.

## Fitur Pasien
- Registrasi akun pasien
- Login pasien
- Dashboard pasien
- Form pendaftaran berobat
- Pilih dokter spesialis
- Pilih tanggal dan jam kunjungan
- Melihat status pendaftaran

## Fitur Admin
- Login admin
- Dashboard admin
- Kelola data pasien
- Kelola data dokter
- Verifikasi pendaftaran pasien
- Kelola jadwal pasien
- Laporan pendaftaran

## Teknologi yang Digunakan
- PHP
- CodeIgniter 3
- MySQL
- Bootstrap 5
- HTML, CSS, JavaScript

## Struktur Login
localhost/aksara-medika              → halaman awal pasien / landing page / home
localhost/aksara-medika/login        → login admin
localhost/aksara-medika/pasien/login → login pasien
localhost/aksara-medika/pasien/register → registrasi pasien
localhost/aksara-medika/dashboard    → dashboard admin
localhost/aksara-medika/pasien/dashboard → dashboard pasien
