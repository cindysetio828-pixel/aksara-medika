# Aksara Medika

Sistem Pendaftaran Pasien Online berbasis web yang dibangun menggunakan CodeIgniter 3 dan MySQL. Project ini dibuat untuk memenuhi tugas UAS mata kuliah Pemrograman Web II.

## Deskripsi Project

Aksara Medika merupakan aplikasi pendaftaran pasien online yang bertujuan mempermudah proses pendaftaran berobat di rumah sakit. Melalui sistem ini, pasien dapat melakukan registrasi akun, mengisi formulir pendaftaran berobat, memilih dokter spesialis, menentukan jadwal kunjungan, serta memantau status pendaftaran secara online.

Admin dapat mengelola data pasien, data dokter, jadwal kunjungan, verifikasi pendaftaran, dan menghasilkan laporan pendaftaran pasien.
